## JAVAWEB后端快速开发结构
## BASE
1. Spring Boot
2. Spring
3. Myabtis
4. Mybatis Plus
4. Swagger 2
5. Lombok
## START
1.idea 导入项目
2.配置CodeGenerator.java,运行生成controller、service、mapper、model
3.完成自己项目的开发
## MODEL
1. 所有service可声明于BaseController中，实现子类controller无需声明service即可调用
2. 所有mapper可声明于DbContext中，实现各个层可直接操作mapper实现快速db操作
